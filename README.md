### **`still in progress`**

### **what is this?**

a simple automation tool for sending, receiving, and managing whatsapp events using the baileys web api.

### **prerequisites**

- node.js (latest lts recommended)
- ffmpeg
- git

### **supported platforms**

- windows
- linux
- macos

### **installation**

```bash
git clone https://github.com/AstroX11/whatsapp-bot
```
